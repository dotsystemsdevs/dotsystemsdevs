## Hello World!

### Open Source

- [**Vibecoders Toolkit**](https://github.com/dotsystemsdevs/vibecoders-toolkit) — Ship apps from zero: MVP, vibecoding, testing, launch, marketing, tracking
- [**app-icon-resizer**](https://github.com/dotsystemsdevs/app-icon-resizer) — CLI to generate Android & iOS app icons from a single PNG
